## Description

This project was developed during the **Nuit de l'Info 2024**. The goal is to educate users about the complexities and impacts of High Technology. By leveraging the **NIRD** method, we provide an interactive and gamified experience that demystifies tech concepts.

The application features a variety of mini-games and simulations, ranging from a Linux installation simulator to classic arcade games with an educational twist.

## The NIRD Method

The **NIRD** method stands at the core of this project. It focuses on:
*   **N**umerique.
*   **I**nclusif.
*   **R**esponsable.
*   **D**urable.

## Features

- **Interactive Dashboard**: A central hub to navigate through different modules.
- **Gamified Learning**:
    - **Snake**: A classic game reimagined.
    - **Wordle**: Guess the tech-related terms.
    - **Quiz**: Test your knowledge on High Tech.
    - **Memory**: Match the concepts.
    - **Relword & Reco**: Word association and recognition games.
- **Simulations**:
    - **Linux Installer**: Experience the process of installing an open-source OS.
- **Visual Experience**: High-quality animations and UI components (Shiny Text, Glare Effects, Pixel Transitions).
- **Multiplayer Features**: Real-time interactions powered by WebSockets.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS.
- **Backend**: Node.js, ProgressJS.
- **Tools**: pnpm, PostCSS.

## Installation

This project is divided into a `client` and a `server`.

### Prerequisites

- Node.js (v16+)
- pnpm

### Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/0xS4cha/nuit2l-info.git
    cd nuit2l-info
    ```

2.  **Install Client Dependencies:**
    ```bash
    cd client
    pnpm install
    ```

3.  **Install Server Dependencies:**
    ```bash
    cd ../server
    pnpm install
    ```

## Usage

### Running the Client
```bash
cd client
pnpm run dev
```

### Running the Server
```bash
cd server
pnpm run start
```
